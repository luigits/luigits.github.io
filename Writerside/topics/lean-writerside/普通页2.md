## inline-frame

这是一个让人惊喜的元素，它可以将外部网页直接嵌入，输入宽高可以调整，单位 (px) 必须指明

```XML

<inline-frame src="https://www.163.com/" width="800px" height="800px"/>
```

## primary-label

这是一个好东西，可以为每一页增加一个类似页眉的效果，这样可以给右侧大纲和页面进行各种简短的标记

`<primary-label>` 标记是一个相对特殊的标记，因为使用它就需要在 <u>labels.list</u> 文件中注册 label


<tabs>
<tab title="<primary-label>">

```XML

<primary-label ref="ab"/>
```

标记元素中的 ref 属性是 <path>labels.list</path> 中 label 标记的 id
</tab>
<tab title="labels.list">

```XML

<primary-label id="ab" name="label" short-name="topic">悬停name上显示的文字</primary-label>
<secondary-label id="beta" name="β" color="tangerine">Beta</secondary-label>
<secondary-label id="2023.3" name="2023.3" color="blue">New in version 2023.3</secondary-label>
```

secondary-label 与 primary-label 相比少了 short-name 和 href 两个属性，其他都是一样的
<deflist>
<def title="name">给页面中的标题增加 label</def>
<def title="color">给 label 施加主题色</def>
<def title="short-name">右侧大纲对应标题左侧增加 label</def>
<def title="href">点击页面标题上的 label 跳转到相应网址</def>
</deflist>
</tab>
</tabs>

## shortcut

使用快捷键的方式需要创建一个与 <path>writerside.cfg</path> 同级的 XML 文件，

我的选择是创建一个名为 <path>keymap.xml</path> 的文件
<tabs>
<tab title="keymap.xml">

```XML
<?xml version="1.0" encoding="UTF-8"?>
<keymap id="my_keymap">
    <layouts>
        <layout name="Windows" platform="PC"/>
        <layout name="macOS" platform="MAC"/>
    </layouts>
    <actions>
        <action id="$Copy">
            <description>Copy</description>
            <shortcut layout="Windows">Ctrl+C</shortcut>
            <shortcut layout="macOS">⌘ C</shortcut>
        </action>
        <action id="$Paste">
            <description>Paste</description>
            <shortcut layout="Windows">Ctrl+V</shortcut>
            <shortcut layout="macOS">⌘ V</shortcut>
        </action>
    </actions>
</keymap>
```

</tab>
<tab title="使用">

```markdown
Press <shortcut key="$Copy"/>to copy.

Press <shortcut key="$Paste"/>to paste.
```

Press <shortcut key="$Copy"/>to copy.

Press <shortcut key="$Paste"/>to paste.

在页面 banner 右侧会出现下拉列表可以切换快捷键平台

如果直接使用 `<shortcut>` 制定了快捷键，在切换平台时无效

Press <shortcut>Ctrl</shortcut><shortcut>V</shortcut> to paste.
</tab>
</tabs>

## tldr

该块将呈现在其所在标题的最开始，标题的正下方。

记录关于主题或章节在关于本页的相关毋庸置疑的内容：例如描述功能呢的快捷键、设置的路径、本章节探讨的先决条件以及其他说明示例

```XML

<tldr>
    <p>
        <ui-path>File | Save</ui-path>
        or
        <shortcut>Ctrl + S</shortcut>
    </p>
    <p>如果文件已在另一个应用程序中打开，则不能保存</p>
</tldr>
```

<tldr>

## tooltip

鼠标悬停时的术语提示，该提示需要在 <ui-path>cfg</u-path> 中创建一个 xml 文件

<tabs>
<tab title="glossary.xml">

```xml
<?xml version="1.0" encoding="utf-8" ?>
<!DOCTYPE terms SYSTEM "https://resources.jetbrains.com/writerside/1.0/glossary.dtd">
<terms>
    <term name="GUI">Graphical User Interface</term>
    <term name="HTTP">Hypertext Transfer Protocol</term>
</terms>
```

</tab>
<tab title="tooltip">

```markdown
Choose your <tooltip term="GUI">GUI</tooltip>.
```

## table

table 的标签语法不如 markdown 语法来的简单快捷，

Writerside 也提供了可视化创建方式 <shortcut key="$idea-insert"/>

使用 <shortcut key="$idea-insert"/> 后在打开的菜单项中选择表，然后就可以使用鼠标来控制创建的表格行列数了。

在某些时候，我们需要让表头凸显，甚至第一列也凸显。Writerside 的 style 为我们提供了选择

</tab>

|---|---|---|
