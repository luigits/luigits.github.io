<snippet id="cli-lib-scoop">

Scoop 是一款专为 Windows 设计的命令行软件包管理工具，它能让你像 Linux 系统一样通过命令快速安装、更新和卸载软件。
其核心优势包括：

- **无需管理员权限**：默认安装在用户目录，避免权限问题。
- **绿色便携化**：软件独立存放，不污染系统注册表。
- **依赖自动处理**：自动配置环境变量和依赖项（如 Java、Python）。
- **海量软件仓库**：支持主流开发工具、实用小软件甚至 GUI 应用。
