<!-- 实例与 instance -->
<snippet id="explain-instance">

<deflist>
<def title="instance">
<p>在 Writerside 中，instance 是对某一类文档集合的名称，Writerside 通过不同的 instance 来管理不同的文档。</p>
<p>instance 翻译后是实例，会与编程中的对象实例产生歧义，因此在指代 Writerside 实例时仍使用 instance。</p>
</def>
</deflist>
</snippet>
