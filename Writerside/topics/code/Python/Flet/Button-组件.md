# Button 组件

Button 支持多种样式、颜色、事件处理程序以供用户交互，并可用于显示文本、图标等。

> 所有组件 (Button) 与子组件 (OutlinedButton) 它们的效果相同，只有外观上的区别

## 属性

| 属性         | 说明           |
|------------|--------------|
| autofocus  | 自动聚焦         |
| bgcolor    | 按钮的背景色       |
| color      | 按钮的颜色        |
| content    | 按钮的名称        |
| elevation  | 按钮的高度        |
| icon       | 按钮内部的图标      |
| icon_color | 图标的颜色        |
| style      | 按钮的样式        |
| url        | 单击按钮时打开的 URL |

## 回调函数

| 属性            | 说明         |
|---------------|------------|
| on_blur       | 当按钮失去焦点时调用 |
| on_click      | 当按钮被点击时调用  |
| on_focus      | 当按钮获得焦点时调用 |
| on_hover      | 当按钮悬停时调用   |
| on_long_press | 长时间按下按钮时调用 |
