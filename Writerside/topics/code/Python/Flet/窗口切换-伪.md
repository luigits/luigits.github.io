# 窗口切换 (伪)

这里说的窗口切换其实是一种掩耳盗铃式的：在主窗口中通过重新注册组件来实现不同内容的显示和功能的展示。

一般来说一个程序会有多个窗口 (例如 VsCode 的**关于**窗口，<ui-path>帮助 | 关于</ui-path>打开)，且子窗口若不关闭可能无法操作主窗口。

不过在现在这个内存容量没有之前珍贵的背景下就无所谓了~~虽然我也不知道这两种方式哪种省内存~~

简单来说有四种方式

- 使用 visible 属性
- 控制 page.controls 列表
- 重新注册组件
- 使用路由

## 使用 visible 属性

制作小工具使用此方法会好用一些，组件需要全部注册

默认显示的界面组件需要修改非默认显示组件 visible 的值为 False

```Python
def on_change(e: ft.Event):
    if e.control.value:
        show1.visible = False
        show2.visible = True
    else:
        show1.visible = True
        show2.visible = False
    page.update()

show1 = ft.Column(
    controls=[
        ft.Text(value="Hello"),
        ft.Text(value="World"),
    ],
)
show2 = ft.Column(
    controls=[
        ft.Text(value="Hello"),
        ft.Text(value="Flet"),
    ],
    visible=False,
)
switch = ft.Switch(
    value=False,
    label="switch controls",
    on_change=on_change,
)
page.add(show1, show2, switch)
```

## 控制 page.controls 列表

这种方式相对触及到了底层实现，仅注册默认显示组件，无需修改组件属性，但需要修改 page.controls

```Python
def on_change(e: ft.Event):
    if e.control.value:
        page.controls = [show2]
        # show1.visible = False
        # show2.visible = True
    else:
        page.controls = [show1]
        # show1.visible = True
        # show2.visible = False

    page.controls.append(switch)

show1 = ft.Column(
    controls=[
        ft.Text(value="Hello"),
        ft.Text(value="World"),
    ],
)
show2 = ft.Column(
    controls=[
        ft.Text(value="Hello"),
        ft.Text(value="Flet"),
    ],
)
switch = ft.Switch(
    value=False,
    label="switch controls",
    on_change=on_change,
)
page.add(show1, switch)
```

## 重新注册组件 (推荐)

在方式 2 的基础上进一步改进，使用 page 对外开放的方法重新注册组件  

```Python
def on_change(e: ft.Event):
    page.clean()
    if e.control.value:
        page.add(show2)
    else:
        page.add(show1)
    page.add(switch)

show1 = ft.Column(
    controls=[
        ft.Text(value="Hello"),
        ft.Text(value="World"),
    ],
)
show2 = ft.Column(
    controls=[
        ft.Text(value="Hello"),
        ft.Text(value="Flet"),
    ],
)
switch = ft.Switch(
    value=False,
    label="switch controls",
    on_change=on_change,
)
page.add(show1, switch)
```
