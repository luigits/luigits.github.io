## 第一个界面

```python
import flet as ft

def main(page: ft.Page):
    # 窗口标题
    page.title = "flet 的标题"
    # 窗口的宽高
    page.window.width = 200
    page.window.height = 300
    # 窗口背景色
    # 官方使用的是定义好的常量，但个人推荐使用 HEX 色值
    page.bgcolor = "#ff00ff"
    # 对齐方式
    page.horizontal_alignment = ft.CrossAxisAlignment.START
    page.vertical_alignment = ft.MainAxisAlignment.START

if __name__ == "__main__":
    ft.run(main=main)
```

> theme 本质上就是各种组件背景色的组合
>
> 因此同时使用 bgcolor 和 theme，theme 会被挤掉
>
> flet 自带的 theme
> - ft.ThemeMode.SYSTEM # 跟随系统
> - ft.ThemeMode.LIGHT # 亮色模式
> - ft.ThemeMode.DARK # 暗色模式
