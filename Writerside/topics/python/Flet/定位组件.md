# 定位组件

在 GUI 编程中，为组件设置唯一标识（如 Flet 的 `key`、Web 的 `id` 等）本质上是提供一种**运行时组件寻址机制**。
它的核心作用类似于数据库的主键或网页中的 `id`，主要解决一个问题：**如何在代码运行过程中精准地找到、操作或追踪某个 UI 元素**。

```Python
page.add(
    ft.Text(value="Hello World!", key="text1"),
    ft.Text(value="Hello World!!", key="text2"),
    ft.Text(value="Hello World!!!", key="text2"),
    ft.Text(value="Hello World!!!!", key="text3"),
)

page.controls[0].value = "HELLO"  # type: ignore # 使用下标的方式获取指定组件

# flet提供给开发者的组件标识符是允许重复的，当需要批量修改某写组件的属性时可以通过此方式设置
# flet的组件唯一标识符由系统提供，不允许修改
for item in page.controls:
    if item.key == "text2":
        item.value = "WORLD"  # type: ignore
```

常用的定位组件方式分为两种：使用变量、使用标识符。
但除非需要批量展示、设置，否则采用变量的方式即可。
