# 制作安装包

<secondary-label ref="global-construct"/>

Python 有独立的编译软件，实际上编译程序与制作安装包还是有一定差别的。

在 python 上，编译软件是将所有需要的库结合在一起，制作为一个可执行文件。

制作安装包则是将软件分开，整个目录缺一不可，否则无法正常运行软件的所有功能
